# Royal Mart

A responsive business website for:

- Royal Talbina
- Bike rental
- Cab booking

## Preview locally

From this folder, run:

```powershell
python -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## Deploy to Vercel

Import this folder or its Git repository into Vercel. No build command or framework preset is required; the included `vercel.json` handles the service-page routes.

## Before going live

1. Add the final price list for Talbina, bikes and cabs.
2. Add the final Talbina pack sizes, ingredients, flavours and delivery charge.

Until a WhatsApp number is configured, submitted requests are stored only in the visitor's browser so the prototype can be tested safely.
